# Smoothie Heaven Website

## Project Description

Smoothie Heaven is a multi-page website for a fictional smoothie shop. The website showcases a variety of smoothies, provides information about the brand, and allows users to view the menu and contact details.

The goal of this project was to practice building a structured website using HTML and CSS with multiple pages and a consistent navigation system.

## Features

* Responsive navigation bar with links to all pages
* Home page introducing the smoothie shop
* Menu page with smoothie categories and prices
* About page explaining the mission of the brand
* Contact page with email, phone number, and location
* Smoothie cards with images, descriptions, and prices
* Hover effects and simple animations for better user interaction

## Technologies Used

* HTML5
* CSS3
* Google Fonts

## Website Pages

* **Home (index.html):** Introduction to Smoothie Heaven and popular smoothies.
* **Menu (menu.html):** Displays fruit, protein, and detox smoothies with images and prices.
* **About (about.html):** Information about the smoothie shop and its mission.
* **Contact (contact.html):** Provides contact details for the business.

## Styling

The website uses CSS to create:

* A flexbox layout for menu cards and navigation
* Smooth hover animations
* A colorful theme to match the smoothie brand
* Styled buttons and menu cards
